# Admin Guide

## Purpose

The admin dashboard is for organizers only.

## Login

Use the values from `.env`:

- `ADMIN_EMAIL`
- `ADMIN_PASSWORD`

## Capabilities

- View teams
- View solves
- Inspect traffic logs
- Toggle chamber availability
- Reset chamber flags
- Review scoreboard state

## Caution

The challenge endpoints themselves are intentionally vulnerable. Do not treat those weaknesses as bugs to patch during competition.
