# Player Guide

## Goal

Clear all seven chambers and collect the static flags.

## Flow

1. Log in with a team account.
2. Open the temple map.
3. Enter chambers in any order, though the intended sequence is strongest.
4. Use the chamber tools to interact with the vulnerable surfaces.
5. Submit solved flags through the flag form.

## Hints

- Treasury is about enumeration, IDOR, and second-order effects.
- Mirror is about SSRF bypasses and internal recon.
- Oracle rewards template-engine fingerprinting and sandbox escape.
- Hydra is about concurrency and duplicated redemption.
- Archive requires format reversal and object-graph abuse.
- Whispers is about stored XSS and bot interaction.
- Core combines all prior artifacts.
